前言

接下去的几个关卡涉及到了http头部的一些内容，我们需要简单的了解一下这些http头部字段。
详细介绍

User-Agent

浏览器表明自己的身份（是哪种浏览器）;

Referer

浏览器向 WEB 服务器表明自己是从哪个 网页/URL 获得/点击 当前请求中的网址/URL；

Cookie

指某些网站为了辨别用户身份、进行 session 跟踪而储存在用户本地终端上的数据（通常经过加密）；
